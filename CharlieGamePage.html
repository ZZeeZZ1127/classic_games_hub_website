mini Wordle

Last Update: 6/2/2025

Lastest Version: alpha

Charlie Ma

Wordle is a word guessing game where the player has six attempts to guess a secret five-letter word. After each guess, the game provides color-coded feedback: green letters are correct and in the right place, yellow letters are correct but in the wrong place, and black letters are incorrect. The game ends when the player guesses the word or uses all attempts.

It has been a daily hobby for me since freshman year. It is easier than other word games but still very fun. It provided a way for me to improve English and spend time. For final project of AP CSA, I wanted to recreate a familiar, fun challenge like Wordle while learning how to structure a Java project, handle user input, and use visual elements like color and layout to provide interactive feedback. This project helped me apply object-oriented programming concepts and improve my skills through hands-on experimentation.
